ENHANCEMENTS / OTHER EXTENSIONS / PROPOSED HTML 3.0 TAGS

table

starts/ends table

Supported:

border, width, cellspacing, cellpadding, caption, bgcolor

Use the TABLE directive to export width and caption.

CELLSPACING equals the cell's left border width

CELLPADDING equals the cell's left margin

td, th

table data /

table heading

Supported:

colspan, rowspan, align, width, valign, bgcolor

Not Supported:

nowrap

Use table cell styles to export width of each column.

TH is never exported. Instead the "<TD><B>" combination is used.

tr

starts a new table row
